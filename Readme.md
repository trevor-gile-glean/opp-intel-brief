# Pro Intel for Sales Skills

This repository is a collection of related Glean skills for sales opportunity intelligence and deal execution workflows.

The skills in this repo are intended to be used together by an agent to research opportunities, synthesize account and deal context, prepare sales teams for customer interactions, and recommend concrete next steps.

## Skills

- [`opp-intel-brief`](./skills/opp-intel-brief/) — Generates an Opportunity Intelligence Brief for Salesforce opportunities, including competitive insights, differentiation messaging, stakeholder context, MEDDPICCS gap analysis, customer references, solution mapping, industry intelligence, and deal advancement recommendations.

## Glean import

Import individual skills from their subtree paths. For example:

```text
https://github.com/trevor-gile-glean/opp-intel-brief/tree/main/skills/opp-intel-brief
```

Use branch or tag URLs rather than commit permalinks so Glean can keep imported skills synced from GitHub.
